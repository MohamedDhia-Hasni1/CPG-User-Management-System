# User Management System 

A comprehensive User Management System built during my software engineering internship at Compagnie des Phosphates de Gafsa (CPG). This platform provides secure employee data management with role-based access control.

##  Project Overview

This system was developed to streamline user management operations within CPG, offering a modern web interface for managing employee data, roles, and permissions securely and efficiently.

##  Internship Context

**CPG (Compagnie des Phosphates de Gafsa)**  
Gafsa, Tunisia  
**Software Engineer Intern** | July 2025

##  Technical Stack

### Frontend
- **Angular** - Modern web framework for the user interface
- **TypeScript** - Type-safe JavaScript development
- **HTML5/CSS3** - Responsive and accessible UI components

### Backend
- **NestJS** - Scalable server-side application framework
- **Prisma** - Modern database ORM and migrations
- **MySQL** - Relational database for secure data storage
- **HeidiSQL** - Database management and administration

### Development Tools
- **HTTPie** - Quick API testing and debugging
- **Postman** - Comprehensive API testing and documentation

##  Key Features

- **User Management** - Complete CRUD operations for employee records
- **Role-Based Access Control** - Granular permissions system
- **Secure Authentication** - Protected access to sensitive data
- **Responsive Interface** - Optimized for various devices and screen sizes
- **Data Validation** - Robust input validation and error handling
